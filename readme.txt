   [Support Vector Machines (SVMs) Tutorial]

This is a simple tutorial for machine learning beginners to classify the make_blobs dataset into two separate classes with a decision boundary called the hyperplane.
  [Before trying this tutorial, machine learning beginners should have the following:]
- Python 2.7 or Python 3.7
- Integrated Development Environment (IDE) i.e Pycharm
- Pandas, Numpy, Matplotlib and ofcourse Scikit-Learn
